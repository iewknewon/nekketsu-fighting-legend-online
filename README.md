# 热血格斗传说 - 浏览器联机版

当前仓库的主要可用交付物已经切换为：

- 你部署一次
- 玩家直接打开网页链接
- 玩家不需要安装模拟器客户端

直接启动：

```bash
npm run online
```

如果你是在 Windows 上只想先快速看一下能不能用，也可以直接双击：

```text
deploy-windows.bat
```

默认入口：

```text
http://localhost:3001/
```

可选检查：

```bash
npm run online:smoke
npm run online:room-smoke
npm run online:advanced-smoke
npm run online:password-smoke
```

说明：

- `online.html` + `online-server/server.js` 是当前主线
- `vendor/emulatorjs` 是固定在仓库里的浏览器模拟器前端
- `src/` 下的 Phaser 代码仍然保留，但不是当前联机交付主线

## 🎮 关于游戏

**热血格斗传说 (熱血格闘伝説)** 是 Technos Japan 于1992年发行在 Famicom 上的格斗游戏，是热血系列的经典作品之一。

### 游戏特色
- ✅ 4人同屏对战
- ✅ 自由混战或2v2团队模式
- ✅ 8个独特角色，各有特色
- ✅ 智能AI对手（3个难度等级）
- ✅ 连击系统和暴击判定
- ✅ 丰富的视觉特效
- ✅ 竞技场格斗玩法

## 🚀 开始使用

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

游戏将在浏览器中运行：http://localhost:3000

### 构建生产版本
```bash
npm run build
```

## 🎮 操作说明

### 玩家1
- **WASD**：移动
- **J**：拳击
- **K**：踢腿
- **J+K**：跳跃

### 玩家2
- **方向键**：移动
- **数字键1**：拳击
- **数字键2**：踢腿
- **1+2**：跳跃

### 通用
- **ESC**：暂停/返回菜单
- **Enter**：确认
- **H**：查看帮助（标题画面）
- **数字键1-4**：快速选择角色

## 💡 游戏技巧

- 连续攻击可以形成连击
- 连击超过3次造成暴击伤害
- 受击后有1秒无敌时间
- 跳跃可以躲避攻击
- 血量低时AI会撤退
- 利用击退效果拉开距离
- 绕圈移动寻找攻击机会
